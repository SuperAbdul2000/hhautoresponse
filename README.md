Приложение для автоокликов на hh.ru по заданным параметрам
Вход доступен только по номеру телефона + пароля (номер телефона без +7)
Параметры поиска настраиваются в /app/config.py
На вакансии с обязательным сопроводительным берётся сопроводительное из cover_letter.txt
Вакансии которые перенаправляют на страницу с тестом сохраняются в отедлньный файл test_vacancies.txt
Если появилась капча при входе в аккаунт нужно просто ввести её и программа продолжит выполнение
